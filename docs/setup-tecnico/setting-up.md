## 1. Introducción
Bienvenido al equipo. Este documento te guiará paso a paso para instalar todas las herramientas necesarias y estandarizadas que utilizamos en el desarrollo de los proyectos de Lead UPN Trujillo.

## 2. Requisitos Previos
Antes de empezar, asegúrate de tener permisos de administrador en tu equipo para poder instalar el software y una conexión a internet estable.

## 3. Instalación de Herramientas Base

### A. Gestor de Paquetes (pnpm)

En Lead UPN Trujillo utilizamos `pnpm` por su rapidez y eficiencia en el manejo de dependencias.

* **Opción Manual (Windows/Mac/Linux):** Ejecuta en tu terminal:

```bash
npm install -g pnpm

```

### B. Editor de Código (VS Code)

Todo el equipo utiliza **Visual Studio Code** para mantener la consistencia.

1. Descárgalo desde su [sitio web oficial](https://www.google.com/search?q=https://code.visualstudio.com/).
2. **Extensiones clave** para nuestro flujo de trabajo:
<!--Axtualizar las extensiones-->
* **ESLint:** Para encontrar y arreglar problemas en el código JavaScript/TypeScript.
* **Prettier:** Para formatear el código automáticamente.
* **Tailwind CSS IntelliSense:** Para el autocompletado de clases.
* **Astro:** Soporte oficial para componentes `.astro`.


## 4. Siguientes Pasos

Una vez tengas todo instalado:

1. Clona el repositorio del proyecto en el que vas a trabajar.
2. Abre la carpeta del proyecto en VS Code.
3. Abre la terminal integrada y ejecuta `pnpm install` para descargar todas las dependencias locales.
